# Pygame 프로젝트

이 프로젝트는 Python의 Pygame 라이브러리를 사용하여 개발되는 게임 또는 인터랙티브 애플리케이션입니다.

## 시작하기

1. Python 3.x가 설치되어 있는지 확인하세요.
2. 아래 명령어로 Pygame을 설치하세요:
   ```bash
   pip install pygame
   ```
3. 프로젝트를 실행하려면:
   ```bash
   python main.py
   ```

## 프로젝트 구조
- `main.py`: 게임의 진입점
- 기타 소스 파일 및 리소스 폴더
